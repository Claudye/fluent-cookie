Fluent-cookie / Documentation

Commencer

Fluent-cookie est un gestionnaire de cookie qui vous permettra de gérer les cookies des manières les plus simples possibles. Il apporte un plus à la sécurité de vos cookies. Toutes les données de vos cookies sont chiffrées.
Vous n'avez pas encore télécharger ? Télécharger

Pourquoi Fluent-Cookie ?

Avec la fonction native setcookie de PHP, vous pouvez déjà faire toutes vos différentes opérations sur les cookies, mais imaginez qu'un jour vous avez envie de renommer un cookie existant, le supprimer directement, ou vous voulez savoir quand il a été créé, cela vous demanderait plus de ligne de code que vous ne le pensiez. Essayez voir !

Plus fou encore supposons que vous avez un cookie existant et un jour vous décidez de vérifier quand est-ce que ce cookie vient à terme (s'expire), les fonctions natives de PHP ne permettent pas de faire ces genres de choses. br Avec fluent-cookie c'est très facile et fluide.

Et la sécurité survient

Tous les cookies crées avec Fluent-cookie sont chiffrés, donc il n'est pas question que quelqu'un lit leur contenu, en tout cas pas facilement

Non seulement mais également, lorsqu'un créé est demandé, vous pouvez afficher son contenu sans peur d'être attaqué par la faille XSS à travers ce cookie

Exemple

Créons un cookie du nom test : 

    <?php
    use Ladis\Cookies\Cookies;            
    require_once __DIR__.'/vendor/autoload.php'
    
    Cookie::create('test')
    ->content('Le contenu de mon cookie')
    ->expires(time() +60)
    ->write();
C'est le minimum pour créer un cookie.
Attention ! Sans la méthode write, votre cookie ne sera pas créé

Testez ! 


    echo $_Cookie['test'] ;
    //Affiche un truc du genre GWtc0HXZ4DOn+1PkpbnfQ8xFwG1xWN0Ktp+It2EjgjCwSwPCf+08DTrRZTELO7is=
Si vous savez ce texte kachakouchou, c'est que votre cookie est bien chriffré.

Pour obtenir ce cookie, vous faites 

$cookie = Cookies::get('test');
$cookie est un objet Ladis\Cookies\CookieMetadata qui contient les informations utiles sur le cookie, avec le quel vous pouvez interagir avec les méthodes disponibles.
Par exemple, pour avoir le timesamps de quand est-ce que ce cookie va s'expirer, appellez la méthode getExpires(), Pour avoir le contenu du cookie utiliser la méthode getContent(). Voir cette section pour voir toutes les méthodes disponibles.

Vous avez vu comment créer un cookie et l'obtenir, et si vous voulez le renommer, supprimer, consulter la liste des méthodes disponibles.

Liste des méthodes sur un cookie

Supposons qu'on a créé un cookie avec Cookies::create('test');
et obtient le cookie $cookie = Cookies::get('test');
Les méthodes suivantes sont valables sur le $cookie

Attention ! Rassurez-vous que $cookie n'est pas null
  • getContent()
    Permet d'avoir le contenu déchiffré du cookie ou
  • getExpires()
    Renvoie le timestamp du cookie
  • getPath()
    Renvoie l'url pour laquelle le cookie est valable
  • getDomain()
    Renvoie le domaine au nom duquel le cookie est enregistré
  • getSecure()
    Renvoie le statut de sécurité pour https
  • getHttponly()
    Renvoie le statut de http
  • createdDate()
    Renvoie la date de création du cookie au format "Y-M-D H:m:s"

Liste des méthodes disponibles pour gérer le cookie

  • create(string $name):CookieManager
    Cette méthode prend en paramètre le nom du cookie à créer.
    et renvoie une une instance de Ladis\Cookies\CookieManager; C'est cette classe qui s'occupe de la création du cookie.

    Une fois la méthode create() est appélée, vous pouvez l'enchainer avec les méthodes suivantes

    • name(string $name) Accepte le nom du cookie
    • content(string $content) Accepte le contenu du cookie
    • expires(int $timesamps = 0)Accepete la date d'expiration du cookie
    • path(string $path = "/") Accepte l'url où le cookie enregistré sera valable
    • domain(string $domain = "") Accepte le domaine auquel associé le cookie
    • secured(bool $secure = true) Enregistre si le domaine est sécurisé
    • httpOnly(bool $httponly = true) Rendre httponly à vrai
    Et puis la méthode write():bool pour enregistrer le cookie

  • get(string $cookiename):CookieMetaData
    Retourne un cookie et ses informations via une instance de Ladis\Cookies\CookieMetaData
  • rename(string $old, string $new)
    Retourne un cookie et ses informations via une instance de Ladis\Cookies\CookieMetaData
  • all():array
    Retourne tous les cookies, créés sans ou avec fluent-cookie
  • has(string $name):bool
    Vérifie l'existence d'un cookie
  • remove(string $name): boolhas(string $name):bool
    Supprime un cookie
  • getCookie(string $cookie): ?CookieMetaData
    Renvoie un cookie avec les métadonnées
  • refresh():void
    Actualise les cookies
